遊び方

ゲームの目的

目標の色を見て、RGBスライダーを調整して同じ色を再現しよう！

操作方法

・R（赤）・G（緑）・B（青）のスライダーを動かして色を作ります

・決定ボタンを押してクリア！

評価基準

ステージごとに基準が変わります

・ステージ1-30: ★★★10 / ★★20 / ★30

・ステージ31-60: ★★★8 / ★★15 / ★25

・ステージ61-90: ★★★5 / ★★10 / ★15

・ステージ91-100: ★★★3 / ★★5 / ★10

※★基準以内でクリア、それ以外は失敗

ヒント機能

困ったときはヒントボタンで色の近さメーターを表示できます

メーターで目標色との距離をリアルタイムで確認できます
