```
目標の色を見て、RGBスライダーを調整して同じ色を再現するパズルゲーム！

【特徴】
・100ステージの色合わせチャレンジ
・星評価で腕試し（★1〜3）
・制限時間内にクリアを目指そう
・ヒント機能で初心者も安心

【こんな人におすすめ】

・色彩感覚を鍛えたい人
・脳トレが好きな人
・パズルゲームが好きな人

【遊び方】
1. 目標の色を確認
2. R（赤）・G（緑）・B（青）のスライダーを調整
3. 色の近さメーターで確認しながら調整
4. 制限時間内に決定ボタンを押してクリア！

【星評価】
★★★: 色差5以内（ほぼ完璧）
★★: 色差10以内（良い）
★: 色差15以内（クリア）

RGBの仕組みを楽しく学べる教育的な要素もあります！
色の三原色を理解しながら、楽しくプレイできます。

ステージが進むにつれて難易度が上がり、より正確な色の再現が求められます。
あなたは100ステージすべてクリアできますか？
```

```
このアプリは、RGBスライダーで色を作る教育的なパズルゲームです。

【主な機能】
- 100ステージの色合わせチャレンジ
- 星評価システム（色差による評価）

- 制限時間内のクリアを目指すゲーム性

【広告について】
- インタースティシャル広告: 5ステージ連続クリア後
- リワード広告: ヒント使用時

【教育的要素】
- RGBの仕組みを学べる
- 色彩感覚を鍛えられる
- デザイナーやイラストレーターの学習にも役立つ

テストアカウントは不要です。すべての機能は広告なしでもテスト可能です。
```
