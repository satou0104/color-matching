# サポート

## Color Match - サポートページ

### アプリについて
Color Matchは、目標の色を見てRGBスライダーを調整し、同じ色を再現するパズルゲームです。

### よくある質問

**Q: ゲームの進行状況が消えてしまいました**
A: ゲームデータはデバイス内に保存されています。アプリを削除したり、デバイスを初期化すると進行状況は失われます。

**Q: 広告が表示されません**
A: 新規アプリの場合、広告配信が開始されるまで数時間〜数日かかる場合があります。

**Q: ヒント機能が使えません**
A: ヒント機能は広告視聴後に使用できます。広告が表示されない場合は、しばらく時間をおいてから再度お試しください。

**Q: 色の判定が厳しすぎます**
A: ステージが進むにつれて許容誤差が小さくなります。色の近さメーターを参考に、細かく調整してみてください。

**Q: 色覚に配慮した機能はありますか？**
A: 色の近さメーターに数値表示があり、色だけに頼らずプレイできます。スライダーにも明確なラベルと数値が表示されます。
