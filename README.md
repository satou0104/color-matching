# Color Match 🎨

RGBで色を作る脳トレパズルゲーム

## 概要
目標の色を見て、RGBスライダーを調整して同じ色を再現するパズルゲームです。100ステージで難易度が上がり、色彩感覚を鍛えることができます。

## 特徴
- 100ステージの色合わせチャレンジ
- 星評価で腕試し（★1〜3）
- 制限時間内にクリアを目指す
- ヒント機能で初心者も安心
- 色覚に配慮したアクセシビリティ対応

## 技術スタック
- Capacitor 7
- Vanilla JavaScript
- CSS3
- AdMob広告

## 開発環境
- iOS 14以上
- Xcode
- Node.js

## セットアップ

### 依存関係のインストール
```bash
npm install
```

### iOS同期
```bash
npx cap sync ios
```

### Xcodeでビルド
```bash
npx cap open ios
```
